# 🍕 Pizziko - Desktop Pizza Ordering System

**Pizziko** is a sleek, modern Windows Forms application designed to simplify the pizza ordering process. Built with a focus on clean UI/UX and real-time logic, it allows users to customize every aspect of their pizza—from size and crust to a variety of toppings—providing an instant price calculation and an order summary.

---

## ✨ Features

* **Interactive Menu:** A user-friendly interface with consistent gold and dark-themed aesthetics.
* **Customization Options:**
    * **Size:** Choose between Small, Medium, or Large.
    * **Crust Type:** Options for Thin or Thick crusts.
    * **Toppings:** Multiple selection support (Extra Cheese, Mushrooms, Tomatoes, Onions, Olives, Green Peppers).
    * **Dining Preference:** Toggle between "Eat In" and "Take Out".
* **Dynamic Price Calculation:** Real-time total price updates (in LE) as the user modifies their selection.
* **Smart Validation:** Prevents empty orders by prompting the user to complete their selection before proceeding.
* **Order Summary:** A dedicated confirmation screen showing the final receipt of the order.

---

## 🛠️ Technical Stack

* **Language:** C++
* **Framework:** .NET Framework (Windows Forms)
* **Environment:** Visual Studio
